# FlashDust Next.js Creator Website

Premium black/gold creator website for FlashDust.

## Correct project structure

```txt
app/
  api/
    youtube/
      route.js
    twitch/
      route.js
  globals.css
  layout.jsx
  page.jsx
package.json
next.config.js
README.md
.env.example
.gitignore
```

## Deploy

Recommended: Vercel.

1. Upload this project to GitHub.
2. Import it in Vercel.
3. Add environment variables later.
4. Deploy.

## Environment variables needed later

```env
YOUTUBE_API_KEY=
YOUTUBE_CHANNEL_ID=
YOUTUBE_HANDLE=FlashDust

TWITCH_CLIENT_ID=
TWITCH_CLIENT_SECRET=
TWITCH_CHANNEL=flashdustwastaken
```
